# Poetic Soul

## Project Overview
Poetic Soul is an innovative platform designed to inspire creativity and poetic expression. This project serves as a hub for poets, writers, and art enthusiasts to share their work, collaborate, and engage with a vibrant community.

## Features
- User authentication and profile management
- Poetry submission and sharing system
- Commenting and feedback mechanisms
- Collaborative writing tools
- Rich text editor support for formatting
- Personalized feeds based on user interests

## Technologies Used
- **Frontend**: React, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Deployment**: Heroku / Vercel

## Installation and Setup
To get a local copy of the project up and running, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Adiyashu/poetic_soul.git
   ```

2. Navigate to the project directory:
   ```bash
   cd poetic_soul
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Set up the environment variables:
   Create a `.env` file in the root directory and include the necessary configurations:
   ```
   DATABASE_URL=your_database_url
   JWT_SECRET=your_jwt_secret
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

## Usage
Once the server is running, you can access the application in your browser at `http://localhost:3000`. 

- Register for an account to begin.
- Submit your poetry on the platform and interact with other users.
